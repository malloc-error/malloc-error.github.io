Basic jQuery

Review from class tutorials

Step 1

Connect a jQuery file, like the one hosted by Google, just after your CSS:
<title>My Title</title>
<link rel="stylesheet" href="style.css">
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>

Step 2

Make a new file in Sublime and save it as script.js

Step 3

Hook it up to your html file in the head, after your jQuery link:
<title>My Title</title>
<link rel="stylesheet" href="style.css">
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>
<script src="script.js"></script>

First line in script.js

$(document).ready(function(){
  // your code goes in here
});

Some methods (commands) from class:

$(".myDiv").addClass("className");
$(".myDiv").removeClass("className");
$(".myDiv").toggleClass("className");
$(".myDiv").animate({"top" : 300, "left" : 100}, 500);

  • .hover()

  • .mouseleave()
