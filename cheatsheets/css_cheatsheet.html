Basic CSS

Review from class tutorials

Step 1

Make a new file in Sublime and save it as style.css

Step 2

Hook it up to your html file in the head, just under title:
<title>My Title</title>
<link rel="stylesheet" href="style.css">

General pattern

tagName {
     attribute: value;
}

.className { /* remember to put a dot before class names! */
     attribute: value;
}

Code examples from class demo

/* ex 1: change the color of the h1 headline to red */
h1 {
     color: rgb(233, 65, 80);
}

/* ex 2: change the background color of paragraphs to blue */
p {
     background-color: rgb(23, 165, 255);
}

/* ex 3: change the opacity of an image to 50% */
img {
     opacity: 0.5;
}

/* ex 4: change the width of an image to 300 pixels */
img {
     width: 300px;
}
