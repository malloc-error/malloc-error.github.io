Summary

Create an interactive experience using only HTML elements (no CSS styling nor JavaScript). Explore and exploit the possibilities of the basic HTML tags to thoughtfully design this piece.

Additional Details

Interaction design is not simply a technical exercise in learning the most complicated code possible; even the basic HTML elements offer opportunities to create interactive experiences. While we will soon go on to cover CSS and JavaScript, learning to design the most engaging possible experience with the simplest possible technology is a crucial skill that will serve you well throughout this course and your career.

For this exercise, focus on designing the interactions rather than refining the type. You are not allowed to use CSS or inline styling (basically anything that goes inside a “style” tag or attribute), but you are welcome to explore the visual possibilities offered by tags such as h1, h2, etc., if it adds to the interaction design of the experience.

Images <img> are allowed if they support the interaction, but do not use them to break the no-style rule. For example, do not use images to display text in a different typeface or color.

Hints and Ideas

The link <a> tag will probably be your friend for this assignment, but you are welcome to find other ways of creating interactions.

You can create a short narrative experience if you like, but make it more engaging than simply clicking through pages of text.

An HTML flipbook using text as visual elements is another way to approach the assignment with a more visual focus.

You could also try to simulate fancier interactions you’ve seen elsewhere with your own lo-fi HTML version.

Play with space elements like &nbsp; to roughly position elements
