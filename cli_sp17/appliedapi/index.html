Summary

Based on your research in the last exercise and consultation with the instructor, code a page that creates simple visual feedback responding to data from your chosen API.

Additional Details

This prompt is intentionally open-ended, but you must do something visual (don't just print out text or numbers). Use CSS and JavaScript to tie a data point to color, or position, or size, or to change an image, etc.

Hint

It may help to proceed one step at a time:

1: Get an API key if needed

2: Get the API working in your browser (plug in a URL and get data back)

3: Use getJSON() to bring data from the API into your code, print out with console.log()

4: Use "if" statements (or some other method, if you like) to set color, size, etc. based on the data point you chose. For example, if I picked temperature as my data point, I could make a dot turn blue if the temperature is below 40 degrees, green for 41–80, and red for above 80.
