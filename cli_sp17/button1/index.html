Deliverables

Summary

Make an HTML element into a button that has visual styling for each state (e.g. normal, disabled, focused, pressed). Use JavaScript/jQuery to change the style of another element when the button is clicked.

Additional Details

Create CSS classes, like .active or .focused, to set the visual designs for different states. Then use jQuery to add or remove those classes. Avoid setting visuals with jQuery as much as possible.

Go beyond the in-class tutorial. Come up with a concept or theme for this exercise to make it into a designed experience rather than simply an exercise in switching colors on and off. Think of real-world buttons/switches for inspiration, like a light switch, etc.

Hints and Ideas

Remember to link jQuery itself in your HTML head section like this:
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.1.1/jquery.min.js"></script>

Also remember to include the above jQuery reference before linking your own script.js file.
