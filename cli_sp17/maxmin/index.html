Summary

Using only two CSS properties, create every possible stylistic variation of a div. How many ways can you transform it? How radically can you transform its appearance by varying only two properties (not counting width and height)?

Additional Details

All HTML elements (div, h1, p, &c.) start out as simple, transparent rectangles by default. CSS allows multiple, layered properties to combine with animations and effects in complex ways, but as designers we should not depend on technical tricks to create interesting visual forms. Research and experiment with a variety of CSS properties from class, or ones that you discover online. Find two CSS properties (in addition to width and height) that allow you to transform the blank div the most.

This assignment will be evaluated on the quality/depth of the transformations as well as the number of variations you make. For example, 255 squares each varying by a slight shade of red is not an interesting transformation for this assignment. Think more along the lines of turning a basic rectangular div into a circle or triangle, as we did in class.

If no single experiment seems to yield a spectacular transformation, or only a few variations, you may submit several experiments (linked from your index.html page for this assignment), to show your work and be evaluated as a whole.

Note on color: Since the div has to be visible, you can use one background-color without it counting against your two CSS properties. In this case, every div would have to be the same color. However, if you want to change background-color for different divs in the same experiment, background-color will count as one of the two changing properties. You may also find ways to avoid using background-color entirely.

Shorthand properties like border are allowed, which let you set multiple aspects of the border (e.g. weight, color, and style) using only one property. Search online and see the cheatsheets to find other "bonus" properties.
