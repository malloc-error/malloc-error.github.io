Summary

Building off your knowledge from the previous exercise and our in-class tutorial, use at least one JavaScript if statement to test whether an event (click, hover, etc.) meets a condition (number, location, etc.). If it does trigger the condition, cause some visual feedback to occur. Use a for loop somewhere in your code, for example setting up the page when it loads, or doing something when the if condition is triggered.

Additional Details

Even though we are calling this exercise Button II, please do not just do an update to your previous exercise; come up with a new idea for this sketch that takes advantage of using variables and if checks.

We did a very simple example of counting clicks in class, but see if you can push farther for your exercise to create something you genuinely think is an interesting interaction + visual combo. See below for some examples of other conditions that you can use in if statements instead of, or combined with, counting.

Hints and Ideas

These are just two examples. Please feel free to email or come to office hours if you have questions on how to do something.

You can use jQuery to check if an element has a class attached to it with .hasClass()
Example: if ($(".box").hasClass("active")) { will check to see if the ".box" div also has the class "active"

You can use jQuery to check the position of an element on the page with .offset().left and offset().top
Example: if ($(".box").offset().left < 400) will be true when the ".box" is less than 400px from the left side of the page
